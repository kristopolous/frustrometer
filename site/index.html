frustrometer

Analyze how appropriate your messages are before you send them.

Try it out! Start typing inappropriate things in this box. Really...

Put it on your site

You can include the script on your site and have your <textareas> magically have a frustrometer without any coding whatsoever

Put it in your browser

Install the plugin.

Available on all the respectable browsers and get the frustrometer with you on the go

Use the API

Currently it's unlimited and free. Try it out. Here's some examples:

% curl 'http://frust.me/analyze' \
 --data '{"data":"wtf is this noise?"}'
{
  "score": 0.912888933264532, 
  "snark": "This is a bad idea."
}
